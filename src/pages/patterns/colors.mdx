# Palette

The colours and system is taken from [Radix Colors](https://www.radix-ui.com/docs/colors).
It uses next naming system:

```
{{ type of UI element }}-{{ category }}--{{ state }}
```

For example: `shape-accent--hover` will be used for `:hover` effect of an interactive element.

Used for non-interactive backgrounds.
