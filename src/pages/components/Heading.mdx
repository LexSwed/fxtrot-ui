## Heading

```tsx preview
<Column gap="lg">
  <Heading>Fxtrot UI</Heading>
  <Heading level="2" as="h2">
    React component library for Fxtrot initiatives.
  </Heading>
</Column>
```

## Dense

`dense` prop resets `Heading`'s `margin` (default: `true`). This is useful when rendered within `Column`, where the spacing between elements has to be determined by the `gap` property of the container.
Since in most use cases the layout is expected to be set by the container, the `dense` prop is `true` by default.

```tsx preview
<React.Fragment>
  <Heading dense={false}>Heading 1</Heading>
  <Heading dense={false} level={2}>
    Heading 2
  </Heading>
  <Heading dense={false} level={3}>
    Heading 3
  </Heading>
  <Heading dense={false} level={4}>
    Heading 4
  </Heading>
  <Heading dense={false} level={5}>
    Heading 5
  </Heading>
  <Heading dense={false} level={6}>
    Heading 6
  </Heading>
</React.Fragment>
```
