## Heading

```tsx preview
<React.Fragment>
  <Heading>Heading 1</Heading>
  <Heading level={2}>Heading 2</Heading>
  <Heading level={3}>Heading 3</Heading>
  <Heading level={4}>Heading 4</Heading>
  <Heading level={5}>Heading 5</Heading>
  <Heading level={6}>Heading 6</Heading>

```

## Dense

`dense` prop resets `Heading`'s `margin`. This is useful when rendered within `Column`, where the spacing between elements has to be determined by the `gap` property of the container.

```tsx preview
<Column gap="6">
  <Heading dense>Fxtrot UI</Heading>
  <Heading level="2" as="h2" dense>
    React component library for Fxtrot initiatives.
  </Heading>
</Column>
```
