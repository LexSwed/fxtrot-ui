## Variant

```tsx preview
<Row gap="xl">
  <Tag label="Primary" />
  <Tag variant="outline" label="Outline" />
  <Tag variant="secondary" label="Secondary" />
</Row>
```

## Size

```tsx preview
<Row gap="xl">
  <Tag label="Small (sm)" size="sm" />
  <Tag label="Medium (md)" size="md" />
  <Tag label="Large (lg)" size="lg" />
</Row>
```

## Theming

Use `ThemeProvider` component to change the primary color of the Tag.

```tsx preview
<ThemeProvider theme="orange">
  <Row gap="xl">
    <Tag label="Primary" />
    <Tag variant="outline" label="Secondary" />
    <Tag variant="secondary" label="Secondary" />
  </Row>
</ThemeProvider>
```
