# Popover

```tsx preview
() => {
  const ref = usePopoverRef();
  return (
    <Popover ref={ref}>
      <Button variant="primary">Create</Button>
      <Popover.Content css={{ minWidth: 300 }} side="left" sideOffset={12}>
        <Flex gap="md" flow="column">
          <TextField label="Full name" placeholder="Ahsoka Tano" />
          <Flex gap="sm" main="end">
            <Button
              onClick={() => {
                ref.current.close();
              }}
            >
              Cancel
            </Button>
            <Button variant="primary" css={{ alignSelf: 'flex-end' }}>
              Submit
            </Button>
          </Flex>
        </Flex>
      </Popover.Content>
    </Popover>
  );
};
```
