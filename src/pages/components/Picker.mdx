# Picker

Picker is a component that mimics default `<select>` element.

```tsx preview
<div className="w-[200px]">
  <Picker label="Item index" placeholder="Pick a number">
    {Array(100)
      .fill(null)
      .map((el, i) => (
        <Picker.Item key={i} value={`${i + 1}`} label={`Item ${i + 1}`} />
      ))}
  </Picker>
</div>
```

## Controlled with validation

```tsx preview
() => {
  const [value, setValue] = React.useState('12');
  return (
    <div className="w-[200px]">
      <Picker
        label="Pick item > 10"
        value={value}
        onChange={setValue}
        validity={Number(value) > 10 ? 'valid' : 'invalid'}
        hint={Number(value) > 10 ? '' : 'Oh no what did you do'}
      >
        {Array(20)
          .fill(null)
          .map((el, i) => (
            <Picker.Item key={i} value={`${i + 1}`} label={`Item ${i + 1}`} />
          ))}
      </Picker>
    </div>
  );
};
```

## Variants

Supports same `variant`s as `Picker` component.

```tsx preview
() => {
  const [variant, setVariant] = React.useState('boxed');
  return (
    <div className="w-[200px]">
      <Picker label="Variant" variant={variant} value={variant} onChange={setVariant}>
        <Picker.Item label="Boxed" value="boxed" />
        <Picker.Item label="Underlined" value="underlined" />
        <Picker.Item label="Flat" value="flat" />
      </Picker>
    </div>
  );
};
```
