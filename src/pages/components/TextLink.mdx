## TextLink

`Text` component as `anchor`. Enables additional props, such as `external` to ease the use and security of `target="_blank"`

```tsx preview
<TextLink textStyle="body-sm" href="https://random.dog/">
  Body style
</TextLink>
```

### External

```tsx preview
<Column gap="8" cross="start" css={{ width: '50%' }}>
```
