# Checkbox

```tsx preview
<Column gap="md">
  <Checkbox label="Accept conditions" />
  <Checkbox label="Accept conditions" defaultChecked />
  <Checkbox label="Accept conditions (disabled)" disabled />
  <Checkbox label="Accept conditions (disabled)" disabled defaultChecked />
</Column>
```

## Secondary text

```tsx preview
<Checkbox label="Accept conditions" secondaryLabel="(required)" required />
```

## Stateful

```tsx preview
() => {
  const [checked, setChecked] = React.useState(false);
  return (
    <Checkbox label="Accept conditions" checked={checked} onChange={setChecked} secondaryLabel="(required)" required />
  );
};
```
