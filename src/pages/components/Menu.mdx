## Menu

```tsx preview
<Menu>
  <Button>
    Open menu
    <Icon as={ChevronDownIcon} size="lg" />
  </Button>
  <Menu.List>
    <Menu.Item onSelect={() => window.alert('copy')}>
      <Icon />
      Copy
    </Menu.Item>
    <Menu.Item onSelect={() => window.alert('cut')}>
      <Icon />
      Cut
    </Menu.Item>
    <Menu.Separator />
    <Menu.Item onSelect={() => window.alert('delete')} css={{ color: '$error' }}>
      <Icon as={TrashIcon} color="$error" />
      Delete
    </Menu.Item>
  </Menu.List>
</Menu>
```

You can modify `offset` from the `side` and along the `align`ment.

## With Dialog

```tsx preview
() => {
  const menuRef = useMenuRef();
  return (
    <Menu ref={menuRef}>
      <Button variant="outline">
        <Icon as={ChevronDownIcon} size="lg" />
        More
      </Button>
      <Menu.List>
        <Dialog>
          <Menu.Item
            css={{ color: '$error' }}
            onSelect={(event) => {
              event.preventDefault();
            }}
          >
            <Icon size="md" as={TrashIcon} />
            Delete
          </Menu.Item>
          {(close) => (
            <Dialog.Modal>
              <Dialog.Title level={3}>Delete topic?</Dialog.Title>
              <form
                onChange={(e) => {
                  setTimeout(() => {
                    close();
                    menuRef.current.close();
                  }, 250);
                }}
              >
                <Picker defaultValue="2">
                  <Picker.Item label="Item 1" value="1" />
                  <Picker.Item label="Item 2" value="2" />
                </Picker>
              </form>
            </Dialog.Modal>
          )}
        </Dialog>
      </Menu.List>
    </Menu>
  );
};
```
