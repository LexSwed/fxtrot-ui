## Menu

```tsx preview
<Menu>
  <Button>
    Open menu
    <Icon as={ChevronDownIcon} />
  </Button>
  <Menu.List>
    <Menu.Item onSelect={() => window.alert('copy')}>
      <Icon />
      Copy
    </Menu.Item>
    <Menu.Item onSelect={() => window.alert('cut')}>
      <Icon />
      Cut
    </Menu.Item>
    <Menu.Separator />
    <Menu.Item onSelect={() => window.alert('delete')} className="text-error">
      <Icon as={TrashIcon} />
      Delete
    </Menu.Item>
  </Menu.List>
</Menu>
```

You can modify `offset` from the `side` and along the `align`ment.

```tsx preview
<Menu>
  <Button variant="tonal" gap="sm">
    Open menu
    <Icon as={ChevronRightIcon} />
  </Button>
  <Menu.List sideOffset={12} side="right" align="start">
    {Array(20)
      .fill(null)
      .map((el, i) => (
        <Menu.Item key={i} onSelect={() => window.alert(i)}>
          Item {i}
        </Menu.Item>
      ))}
  </Menu.List>
</Menu>
```

## With Dialog

```tsx preview
() => {
  const menuRef = useMenuRef();
  return (
    <Menu ref={menuRef}>
      <Button variant="tonal" gap="md">
        <Icon as={EllipsisVerticalIcon} />
        More
      </Button>
      <Menu.List>
        <Dialog>
          <Menu.Item
            className="text-error"
            onSelect={(event) => {
              event.preventDefault();
            }}
          >
            <Icon size="md" as={TrashIcon} />
            Delete
          </Menu.Item>
          {(close) => (
            <Dialog.Modal>
              <Dialog.Title level={3}>Delete topic?</Dialog.Title>
              <form
                onChange={(e) => {
                  setTimeout(() => {
                    close();
                    menuRef.current.close();
                  }, 250);
                }}
              >
                <Picker defaultValue="2">
                  <Picker.Item label="Item 1" value="1" />
                  <Picker.Item label="Item 2" value="2" />
                </Picker>
              </form>
            </Dialog.Modal>
          )}
        </Dialog>
      </Menu.List>
    </Menu>
  );
};
```
