# Tabs

A set of layered sections of content—known as tab panels—that are displayed one at a time.

```tsx preview
<Tabs defaultValue="npm">
  <Tabs.List>
    <Tabs.Trigger value="npm">NPM</Tabs.Trigger>
    <Tabs.Trigger value="yarn">Yarn</Tabs.Trigger>
  </Tabs.List>
  <Tabs.Content value="npm">
    <MultilineCode code="npm install @fxtrot/ui" language="bash" lineNumber={false} />
  </Tabs.Content>
  <Tabs.Content value="yarn">
    <MultilineCode code="yarn add @fxtrot/ui" language="bash" lineNumber={false} />
  </Tabs.Content>
</Tabs>
```

## Orientation

Control the orientation of tabs with [Flex](/components/Flex)'s `flow` prop:

```tsx preview
<Tabs defaultValue="npm" flow="row">
  <Tabs.List className="max-w-md">
    <Tabs.Trigger value="npm">NPM</Tabs.Trigger>
    <Tabs.Trigger value="yarn">Yarn</Tabs.Trigger>
  </Tabs.List>
  <Tabs.Content value="npm">
    <MultilineCode code="npm install @fxtrot/ui" language="bash" lineNumber={false} />
  </Tabs.Content>
  <Tabs.Content value="yarn">
    <MultilineCode code="yarn add @fxtrot/ui" language="bash" lineNumber={false} />
  </Tabs.Content>
</Tabs>
```

## URL sync

Often, you want to preserve selected tab between page reloads. There's no better way than using URL Search Params for this.
