# Flex

`Flex` is a parent reusable component for many other primitives, like `Button`, that allows to easily align the content. However, for layouts it is recommended to use [Row](/components/Row) or [Column](/components/Column) as they are faster to understand. Use `Flex` when you need to change content alignment on different screen sizes, using [responsive props](https://stitches.dev/docs/responsive-styles).

## Alignment

Since `Flex` can be used for rows and columns, use `main` and `cross` to align items inside `Flex`
across main axis or cross axis.

`main="start"` (default)

```tsx preview
<React.Fragment>
  <Flex css={{ width: '400px', bc: '$surface--elevated' }} flow="column" gap="md" main="start">
    <ExampleBox width="$16" />
    <ExampleBox width="$12" />
    <ExampleBox width="$20" />
  </Flex>
  <Flex css={{ width: '400px', bc: '$surface--elevated', mt: '$8' }} flow="row" gap="md" main="start">
    <ExampleBox width="$16" />
    <ExampleBox width="$12" />
    <ExampleBox width="$20" />
  </Flex>
</React.Fragment>
```

`main="center"`

```tsx preview
<React.Fragment>
  <Flex css={{ height: '220px', bc: '$surface--elevated' }} gap="md" flow="column" main="center">
    <ExampleBox width="$16" />
    <ExampleBox width="$12" />
    <ExampleBox width="$20" />
  </Flex>
  <Flex css={{ width: '400px', bc: '$surface--elevated', mt: '$8' }} gap="md" flow="row" main="center">
    <ExampleBox width="$16" />
    <ExampleBox width="$12" />
    <ExampleBox width="$20" />
  </Flex>
</React.Fragment>
```

`main="end"`

```tsx preview
<React.Fragment>
  <Flex css={{ width: '400px', height: '240px', bc: '$surface--elevated' }} gap="md" flow="column" main="end">
```

## Flow

You can reverse the components render using `flow="column-reverse"`

```tsx preview
<Flex flow="column-reverse" gap="md">
  <ExampleBox>1</ExampleBox>
  <ExampleBox>2</ExampleBox>
  <ExampleBox>3</ExampleBox>
</Flex>
```

And `row-reverse`

```tsx preview
<Flex flow="row" main="center" cross="end" flow="row-reverse" gap="md">
  <ExampleBox height="24px">1</ExampleBox>
  <ExampleBox height="44px">2</ExampleBox>
  <ExampleBox height="64px">3</ExampleBox>
</Flex>
```

## Wrap

Control content wrapping via `wrap` property:

```tsx preview
<Box css={{ width: 50 * 4 + (4 - 1) * 8 }}>
  <Flex flow="row-reverse" wrap="wrap-reverse" gap="sm">
    {Array(16)
      .fill(0)
      .map((el, i) => (
        <ExampleBox key={i}>{i + 1}</ExampleBox>
      ))}
  </Flex>
</Box>
```
