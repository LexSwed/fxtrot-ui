## Text

```tsx preview
<Text as="p">This is a normal text rendered as paragraph</Text>
```

## Text variants

```tsx preview
<Column gap="sm">
  <Text>Default text</Text>
  <Text tone="accent">Default text</Text>
  <Text tone="light">Light text</Text>
  <Text tone="success">Success text</Text>
  <Text tone="danger">Error text</Text>
</Column>
```

## Alignments

```tsx preview
<Column gap="sm" cross="stretch">
  <Text>Default align</Text>
  <Text align="start">Start align</Text>
  <Text align="center">Center align</Text>
  <Text align="end">End align</Text>
</Column>
```

## Text style

`Text` component doesn't allow to change the size diectly, and instead offers to use `textStyle` prop for consistent styling.

<Column gap="md" cross="stretch">
  {[
    'body-sm',
    'body-md',
    'body-lg',
    'body-xl',
    'label-sm',
    'label-md',
    'label-lg',
    'title-sm',
    'title-md',
    'title-lg',
    'headline-sm',
    'headline-md',
    'headline-lg',
    'mono-sm',
    'mono-md',
    'mono-lg',
    'overline',
  ].map((key) => (
    <TextWithComputedStyle key={key} textStyle={key}>
      {key}
    </TextWithComputedStyle>
  ))}
</Column>

Example:

```tsx preview
<div>
  <Text textStyle="body-md" as="p">
    This is a normal text rendered as paragraph
  </Text>
  <div className="bg-primary-container p-8">
    <Text textStyle="body-md" as="p" className="text-on-primary-container">
      This is a normal text rendered as paragraph
    </Text>
  </div>
</div>
```

You can still change the size through styles, but this is discouraged:

```tsx preview
<Text textStyle="headline-md" className="text-xs">
  Headline, but really small
</Text>
```

## Clamping

You can specify whether to truncate the text on one line or multiple:

```tsx preview
() => {
  const loremIpsum = `
row scuttle parrel provost Sail ho shrouds spirits boom mizzenmast yardarm. Pinnace holystone mizzenmast quarter crow's nest nipperkin grog yardarm hempen halter furl. Swab barque interloper chantey doubloon starboard grog black jack gangway rutters.
Deadlights jack lad schooner scallywag dance the hempen jig carouser broadside cable strike colors. Bring a spring upon her cable holystone blow the man down spanker Shiver me timbers to go on account lookout wherry doubloon chase. Belay yo-ho-ho keelhaul squiffy black spot yardarm spyglass sheet transom heave to.
Trysail Sail ho Corsair red ensign hulk smartly boom jib rum gangway. Case shot Shiver me timbers gangplank crack Jennys tea cup ballast Blimey lee snow crow's nest rutters. Fluke jib scourge of the seven seas boatswain schooner gaff booty Jack Tar transom spirits.
`;
  return (
    <Column gap="md" cross="stretch">
      <div className="h-[1em * 2] w-[300px] overflow-hidden rounded-md bg-secondary-container p-2 text-on-secondary-container">
        <Text lineClamp={1}>{loremIpsum}</Text>
      </div>
      <div className="h-[2em * 2] w-[300px] overflow-hidden rounded-md bg-secondary-container p-2 text-on-secondary-container">
        <Text lineClamp={2}>{loremIpsum}</Text>
      </div>
      <div className="h-[3em * 2] w-[300px] overflow-hidden rounded-md bg-secondary-container p-2 text-on-secondary-container">
        <Text lineClamp={3}>{loremIpsum}</Text>
      </div>
      <div className="h-[4em * 2] w-[300px] overflow-hidden rounded-md bg-secondary-container p-2 text-on-secondary-container">
        <Text lineClamp={4}>{loremIpsum}</Text>
      </div>
      <div className="h-[5em * 2] w-[300px] overflow-hidden rounded-md bg-secondary-container p-2 text-on-secondary-container">
        <Text lineClamp={5}>{loremIpsum}</Text>
      </div>
      <div className="h-[6em * 2] w-[300px] overflow-hidden rounded-md bg-secondary-container p-2 text-on-secondary-container">
        <Text lineClamp={6}>{loremIpsum}</Text>
      </div>
    </Column>
  );
};
```
