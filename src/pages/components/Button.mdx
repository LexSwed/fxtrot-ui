## Variant

Do not use `variant="link"` with actual links, use [`LinkButton`](/components/LinkButton) component instead.

```tsx preview
<Grid gap="lg" cross="center" main="center" columns="repeat(2, auto)" css={{ p: '$4' }}>
  <Button variant="primary">Primary</Button>
  <Button disabled variant="primary">
    Primary (disabled)
  </Button>
  <Button variant="outline">Outline</Button>
  <Button variant="outline" disabled>
    Outline (disabled)
  </Button>
  <Button variant="secondary">Secondary (default)</Button>
  <Button variant="secondary" disabled>
    Secondary (disabled)
  </Button>
  <Button variant="flat">Flat</Button>
  <Button variant="flat" disabled>
    Flat (disabled)
  </Button>
  <Button variant="link">Link</Button>
  <Button variant="link" disabled>
    Link (disabled)
  </Button>
</Grid>
```

## Theming

Use `ThemeProvider` component to change the primary color of the button.

```tsx preview
<ThemeProvider theme="purple">
  <Column gap="xl">
    <Row gap="sm" css={{ px: '$10' }}>
      <Button variant="primary">Primary</Button>
      <Button disabled variant="primary">
        Primary (disabled)
      </Button>
    </Row>
    <Row gap="sm" css={{ px: '$10' }}>
      <Button variant="secondary">Secondary</Button>
      <Button variant="secondary" disabled>
        Secondary (disabled)
      </Button>
    </Row>
    <Row gap="sm" css={{ px: '$10' }}>
      <Button variant="flat">Flat</Button>
      <Button variant="flat" disabled>
        Flat (disabled)
      </Button>
    </Row>
    <Row gap="sm" css={{ px: '$10' }}>
      <Button variant="link">Link</Button>
      <Button variant="link" disabled>
        Link (disabled)
      </Button>
    </Row>
  </Column>
</ThemeProvider>
```

## With elements

If you only want to put [`Icon`](/components/Icon) inside the `Button`, use [`IconButton`](/components/IconButton) component instead.

```tsx preview
<Column cross="start" gap="xl" css={{ p: '$8' }}>
  <Button variant="primary" gap="2">
    <Icon as={DotsVerticalIcon} />
    More
  </Button>
</Column>
```
