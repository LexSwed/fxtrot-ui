## Variant

Do not use `variant="link"` with actual links, use [`LinkButton`](/components/LinkButton) component instead.

```tsx preview
<Grid gap="lg" cross="center" main="center" columns="repeat(2, auto)" css={{ p: '$4' }}>
  <Button variant="primary">Primary</Button>
  <Button disabled variant="primary">
    Primary (disabled)
  </Button>
  <Button variant="outline">Outline</Button>
  <Button variant="outline" disabled>
    Outline (disabled)
  </Button>
  <Button variant="tonal">Tonal</Button>
  <Button variant="tonal" disabled>
    Tonal (disabled)
  </Button>
  <Button variant="flat">Flat</Button>
  <Button variant="flat" disabled>
    Flat (disabled)
  </Button>
  <Button variant="link">Link</Button>
  <Button variant="link" disabled>
    Link (disabled)
  </Button>
</Grid>
```

## Icon Button

Icon Button allows to easily set up a Button with [Icon](/components/Icon) only as a child.

```tsx preview

  <Row gap="md" className="px-10">
    <Button icon={ChevronRightIcon} aria-label="Move right" size="xs" variant="primary"></Button>
```
